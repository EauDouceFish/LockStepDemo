新引擎第一次被组装成"能吃真实角色文件并逐帧跑"的东西：

```
Assets/Logic/Mugen/Battle/

  MCharLoader    从 CNS/CMD/AIR 文本装配 MCharData；SpawnChar 建运行期 MChar
  MBattleEngine  每帧 pipeline = 命令→状态机→物理→动画(M8)→命中
  MPhysics       位置积分 + 摩擦/重力（见 1.2）
Assets/Logic/Mugen/Parse/MDefParser   解析 .def [Files] 段取文件名
```

**MBattleEngine.Tick(inputs) 每帧顺序**（对齐 CLAUDE §8.3）：
1. `CommandList.Update`：压入本帧输入 + 命令匹配（搓招）
2. `MStateMachine.RunFrame`：负状态/当前状态控制器、ChangeState 同帧重入
3. `MPhysics.Step`：位置积分 + 摩擦/重力
4. `MAnimSystem.Action`：动画推进（M8）+ 派生当前帧 Clsn
5. `MHitSystem.TryHit`（双向，仅 2 角色）：命中检测与结算

`ComputeHash()` 覆盖全角色 → 接快照/回滚（M10）。**无静态可变态**，运行态全在 MChar 上。
